# 🚗 Car Security System

A smart vehicle security and monitoring system designed to detect threats and nearby obstacles, enhancing safety and preventing parking-related issues.

---

## 🔍 Overview
This project focuses on improving vehicle safety by detecting objects or threats within a defined radius around the car. When a vehicle or object comes too close, the system triggers an alert, helping the driver take action and avoid collisions or congestion.

---

## ⚙️ Features
- 🚘 Proximity detection of nearby vehicles  
- 🔊 Alert system (beep/notification) when objects are too close  
- 📏 Configurable detection range (e.g., 8 meters)  
- 🛡️ Enhances vehicle safety and parking efficiency  
- 🔄 Real-time monitoring (based on implementation)

---

## 🧠 Working Principle
1. The system continuously monitors the surroundings of the vehicle  
2. Detects objects or vehicles within a specified radius  
3. Calculates distance using sensors / image processing  
4. If distance < threshold → alert is triggered  
5. Helps driver adjust position or take preventive action
